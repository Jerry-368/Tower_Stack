# Tower Stack

Tower Stack is a simple timing-based stacking game built using Unity.  
The goal is to stack moving blocks as accurately as possible to build the tallest tower.

---

## Gameplay

- Tap or click to drop blocks  
- Align blocks perfectly to maintain size  
- Misalignment reduces block size  
- Game ends when a block completely misses  

---

## Overview

The player must drop moving blocks on top of each other with precise timing.  
As the tower grows, the difficulty increases, requiring better accuracy and focus.
---

## Features

* Smooth and responsive gameplay
* Progressive difficulty
* Score tracking system
* Minimal and clean design

---

## How to Play

1. Click or tap to drop the moving block
2. Align the block with the previous one
3. Misalignment reduces the block size
4. Continue stacking to increase your score
5. The game ends when the block completely misses
